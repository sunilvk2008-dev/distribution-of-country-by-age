# distribution-of-country-by-age
# 📊 Population Distribution Visualization

## 📌 Overview

This project visualizes the distribution of country populations using a histogram. The analysis was performed using the World Bank Population dataset with Python, Pandas, and Matplotlib. The visualization helps understand how population values are distributed across different countries.

---

## 🎯 Objective

The objective of this project is to explore and visualize the distribution of a continuous variable (population) using a histogram.

---

## 📂 Dataset

* **Dataset Name:** World Bank Population Dataset
* **Source:** World Bank Open Data
* **File:** Data_Extract_FromWorld Development Indicators.xlsx

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib

---

## 📈 Project Workflow

1. Load the dataset using Pandas.
2. Remove unnecessary metadata columns.
3. Select the latest available population values.
4. Handle missing values.
5. Create a histogram using Matplotlib.
6. Add chart title, axis labels, and grid.
7. Save the histogram as an image.

---

## 📊 Output

The project generates a histogram showing the distribution of country populations. This visualization helps identify common population ranges and outliers.
