# 📊 Population Distribution Visualization

## 📌 Project Overview

This project visualizes the distribution of the male and female population using data from the World Development Indicators dataset. Python is used to process the data and generate histograms that help understand the population distribution.

---

## 🎯 Objective

- Load population data from an Excel file.
- Analyze male and female population data.
- Create histograms for both male and female populations.
- Save the charts as PNG images.

---

## 📂 Dataset

- **Dataset:** World Development Indicators
- **Format:** Excel (.xlsx)

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib

---

## ▶️ How to Run

1. Clone this repository.

2. Install the required libraries:

```bash
pip install pandas matplotlib openpyxl
```

3. Run the Python files:

```bash
python histogram_male.py
```

```bash
python histogram_female.py
```

---

## 📈 Output

The project generates the following visualizations:

- `histogram_male.png`
- `histogram_female.png`

These histograms show the distribution of male and female population data.
